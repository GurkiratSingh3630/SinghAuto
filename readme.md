# SinghAuto 🚗

SinghAuto is a full-stack car service booking web application that allows customers to book vehicle services online and manage their bookings.

## Features

- User registration and login
- Book car services (oil change, brakes, tires, etc.)
- Admin dashboard
- Dynamic UI using Pug templates
- MongoDB database integration

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Pug
- HTML / CSS
- JavaScript

## Installation

Clone the repository

git clone https://github.com/GurkiratSingh3630/SinghAuto.git

Install dependencies

npm install

Start the server

npm start
